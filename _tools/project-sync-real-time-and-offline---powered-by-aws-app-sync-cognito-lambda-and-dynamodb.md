name: Project Sync Real time and offline   Powered by AWS App Sync Cognito Lambda
  and DynamoDB
description: An android application which allows users to add projects and for each
  project user can publish agenda or tasks which is real time to all other users who
  are subscribed to the same project. It shows real time updates and allows offline
  changes to be saved in cache which are updated once connected to a network. It also
  features cognito authentication for user logins
